[Docker Image]() with Stress tool. This image is acting like `stress` command and passes all arguments.

### Build

    docker build -t polinux/stress .

### Basic Usage

    docker run \
      -ti \
      --rm \
      polinux/stress stress \
        --cpu 1 \
        --io 1 \
        --vm 1 \
        --vm-bytes 128M \
        --timeout 1s \
        --verbose
